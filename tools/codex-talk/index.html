코딩 AI와 대화하는 법

클로드코드·코덱스 같은 코딩 AI에게 지금 상황에 맞는 호출문을 골라 복사하세요. 카드를 누르면 언제 쓰는지 · 과할 때 · 복사용 문구가 떠요.

이렇게 쓰세요

1상황에 맞는 카드를 눌러요→ 2‘복사’ 눌러요→ 3쓰던 코딩 AI(클로드코드·코덱스 등)에 붙여넣어요

예) ‘읽기 전용으로만 검토’를 복사 → 채팅 첫 줄에 붙여넣으면 → AI가 함부로 고치지 않고 상태만 봐줘요.

찾는 호출문이 없어요. 다른 단어로 검색하거나 ‘전체’를 눌러보세요.
