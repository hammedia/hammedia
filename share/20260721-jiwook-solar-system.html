1. 별과 행성

별행성
스스로 빛을 냄스스로 빛을 못 냄 (태양 빛을 반사)
지구에서 아주 멀다별보다 훨씬 가깝다
여러 날 봐도 위치가 그대로여러 날 보면 위치가 조금씩 변함
태양, 북극성수성·금성·지구·화성·목성·토성·천왕성·해왕성

스스로 빛나면 별, 빌려서 빛나면 행성

금성·화성·목성·토성은 행성인데도 웬만한 별보다 밝게 보여. 그래서 밝기로는 구분할 수 없어.

2. 태양계 구성원

태양 + 태양의 영향을 받는 천체들 + 그 공간 = 태양계

  • 태양 — 태양계에서 유일하게 스스로 빛을 내는 천체. 지구 에너지의 근원.
  • 행성 8개 — 수성 · 금성 · 지구 · 화성 · 목성 · 토성 · 천왕성 · 해왕성
  • 그 밖에 위성(달 등), 소행성, 혜성

밤하늘에 보이는 다른 별들은 전부 태양계 바깥에 있어.

3. 크기 순서와 거리 순서

태양에서 가까운 순서

수성 → 금성 → 지구 → 화성 → 목성 → 토성 → 천왕성 → 해왕성

수금지화목토천해

크기 순서 (큰 것부터)

목성 → 토성 → 천왕성 → 해왕성 → 지구 → 금성 → 화성 → 수성

작은 행성 (안쪽 4개)큰 행성 (바깥쪽 4개)
수성 · 금성 · 지구 · 화성목성 · 토성 · 천왕성 · 해왕성
가장 작은 것 = 수성가장 큰 것 = 목성

거리 순서와 크기 순서는 서로 다른 두 개야. 예를 들어 화성은 지구보다 태양에서 멀지만, 크기는 지구보다 작아.

4. 태양계 그림 읽기

교과서나 책에 나오는 태양계 그림은 실제 비율이 아니야. 실제로는 행성 사이 거리가 너무 멀어서, 크기와 거리를 동시에 정확하게 그리는 게 불가능하거든.

그림에서 행성들이 옹기종기 모여 있다고 해서 실제로 가까운 게 아니야.

5. 별자리와 북극성

별자리 = 무리 지어 있는 별들을 선으로 이어 이름을 붙인 것

북극성 찾는 법

  • 북두칠성으로: 국자 끝의 두 별 사이 거리를 5배 연장
  • 카시오페이아로: 바깥쪽 두 선을 연장해 만나는 점에서 가운데 별 방향으로 5배 연장

둘 다 5배 — 북극성은 항상 북쪽, 움직이지 않음

북극성은 밤하늘에서 가장 밝은 별이 아니야. 중요한 건 밝기가 아니라 움직이지 않는다는 점이야.
