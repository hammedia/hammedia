1. 수집하는 정보

현재 버전의 InkTrio는 별도의 회원가입을 제공하지 않으며, 앱 서버로 계정 정보를 전송하지 않습니다.

앱 안에서 사용자가 직접 입력하거나 선택할 수 있는 정보는 다음과 같습니다.

  • 만년필 정보
  • 잉크 정보
  • 종이 또는 노트 정보
  • 만년필, 잉크, 종이의 조합 기록
  • 사용감 메모
  • 사용자가 선택하거나 촬영한 조합 사진

이 정보는 현재 버전 기준으로 사용자의 기기 안에 저장됩니다.

2. 사진과 카메라 권한

InkTrio는 사용자가 조합 기록에 사진을 첨부할 수 있도록 사진 라이브러리 접근 권한을 요청할 수 있습니다.

사용자가 선택한 사진은 앱 내부 저장소에 복사되어 조합 기록과 연결됩니다. 사진은 데이터베이스에 직접 BLOB 형태로 저장하지 않고, 앱 내부 파일 경로와 연결해 관리합니다.

앱은 향후 촬영 기능 또는 사진 첨부 흐름을 위해 카메라 접근 권한을 요청할 수 있습니다.

3. 정보 사용 목적

앱에 저장된 정보는 다음 목적에만 사용됩니다.

  • 사용자가 자신의 만년필, 잉크, 종이 조합을 기록하기 위해
  • 저장한 조합을 목록과 상세 화면에서 다시 확인하기 위해
  • 사진이 포함된 기록을 나중에 다시 찾기 위해

4. 외부 공유와 제3자 제공

현재 버전의 InkTrio는 사용자의 기록 데이터를 외부 서버로 전송하거나 제3자에게 판매하지 않습니다.

향후 백업, 동기화, 내보내기, 공유 기능이 추가될 경우 개인정보처리방침을 업데이트하고 필요한 안내를 제공합니다.

5. 데이터 보관과 삭제

사용자가 앱에 입력한 정보와 사진은 사용자의 기기 안에 보관됩니다.

사용자가 앱 안에서 기록이나 사진을 삭제하면 해당 앱 데이터가 삭제될 수 있습니다. 앱을 기기에서 삭제하면 기기 안의 앱 데이터도 함께 삭제될 수 있습니다.

6. 어린이 개인정보

InkTrio는 어린이를 대상으로 한 앱이 아닙니다.
